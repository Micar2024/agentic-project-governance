# Security Policy

This repository is a template project. Do not publish sensitive material here.

## Never include

- Secrets or credentials
- API keys or tokens
- Private repository links
- Internal file paths
- Production configuration
- Customer data
- Financial data
- Confidential company information
- Real incident details without explicit permission

## Reporting a problem

If you find sensitive information in this repository, open a private security report if available, or contact the maintainers through the safest available channel.
