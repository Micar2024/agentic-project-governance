# v0.1.0 Release Notes

Initial draft release of `agentic-project-governance`.

## Included

- Public positioning README
- Generic role model
- Standard workflow
- Safety boundaries
- Task packet template
- PR review packet template
- Acceptance report template
- Fictional minimal example
- GitHub issue and PR templates

## Excluded

- Runtime automation
- CLI tools
- GitHub Actions
- Production integrations
- Private project examples

## Known limitations

This is a template-only release. It does not guarantee code correctness, production readiness, or business correctness.
