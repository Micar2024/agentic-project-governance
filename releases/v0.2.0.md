# v0.2.0 - Usability and review workflow update

## Highlights

- Added three fictional examples for common AI-assisted delivery scenarios.
- Added workflow diagrams for fast vs standard channel decisions.
- Added a template validation guide for task packets, PR review packets, and acceptance reports.
- Added a design review template for user-facing and safety-sensitive changes.

## Added

- `examples/cli-tool-bugfix/` - channel escalation example (fast to standard channel)
- `examples/dashboard-ui-safety-review/` - human approval gate example
- `examples/documentation-sprint/` - multi-task aggregation and release readiness example
- `docs/workflow.md` updates with Mermaid diagrams for channel selection, escalation triggers, and human approval gates
- `docs/template-validation-guide.md` - checklists for task packet, PR review packet, and acceptance report validation
- `templates/design-review.md` - design review template with risk review and AI-agent execution boundaries

## Improved

- README links to new workflow, validation, and design review materials.
- All Markdown files cleaned to ASCII-only to prevent GitHub hidden/bidirectional Unicode warnings.
- Open issues clearly separated into v0.2 completed work and v0.3 candidates.

## Not included

- No runtime, CLI tools, or automation.
- No GitHub Action.
- No JSON schema.
- No enterprise compliance framework.
- No production automation.
- No dependency audit template (planned for v0.3).
- No multi-language translations (planned for v0.3).

## Known limitations

- No external contributors yet.
- No automated validation.
- Schema and GitHub Action planned for v0.3.
- Multi-language translations not included in this release.
- Single fictional example from v0.1.0 (`minimal-agentic-project`) retained as-is.
