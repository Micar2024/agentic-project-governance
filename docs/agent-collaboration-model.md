# Agent Collaboration Model

A safe collaboration model separates planning, implementation, review, and acceptance.

## Separation of duties

- The implementer should not be the only reviewer.
- The reviewer should check scope and evidence, not just style.
- The coordinator should close the loop with an acceptance report.
- The human owner keeps final authority.

## Handoff rule

A handoff must include objective, known facts, completed work, remaining work, blockers, next action, and forbidden actions.
