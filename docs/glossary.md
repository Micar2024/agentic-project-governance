# Glossary

## Task Packet

A structured instruction package that defines objective, scope, forbidden scope, evidence, risks, and rollback.

## PR Review Packet

A review document that summarizes changes, scope compliance, evidence, and residual risks.

## Acceptance Report

A closure document that states what was done, how it was verified, what remains risky, and whether the task passes.

## Fast Channel

A lightweight path for low-risk reversible edits.

## Standard Channel

A stricter path for non-trivial, risky, or externally visible work.
