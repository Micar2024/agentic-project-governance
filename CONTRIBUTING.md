# Contributing

Thank you for considering a contribution.

This repository accepts public, sanitized governance templates and fictional examples for AI-assisted software projects.

## Contribution rules

- Use fictional examples by default.
- Do not submit private project data.
- Do not submit secrets, credentials, internal paths, private repository links, customer data, financial data, production configuration, or confidential business information.
- Keep templates tool-agnostic when possible.
- Include acceptance criteria and verification evidence for non-trivial changes.

## Good contributions

- Clearer templates
- Better checklists
- Safer boundary language
- Fictional examples that demonstrate the workflow
- Documentation improvements

## Avoid

- Vendor-specific hype
- Unverified claims
- Real company workflows without permission
- Prompts that bypass human review
- Automation that implies production approval without humans
